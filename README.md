# Karan Singh Thakur — Spider-Hero Inspired PHP/LAMP Portfolio

This version uses a clean spider-hero inspired theme for stronger visual personality while keeping the portfolio professional and recruiter-friendly.

## Design changes

- Red/blue superhero-inspired visual system
- Animated web-line background
- Cursor glow effect
- Soft swinging hero card
- Web-shield style custom SVG emblem
- Animated project cards and impact cards
- No official Spider-Man logos, characters, or copyrighted artwork
- Compact project cards with no extra blank space

## Run locally

```bash
cd karan_php_lamp_portfolio_spider_theme
php -S localhost:8000
```

Open:

```text
http://localhost:8000
```

Optional PHP OOP sample:

```text
http://localhost:8000/assets/php/oop-sample.php
```

## Deploy

Upload to GitHub Pages, Netlify, Vercel, cPanel `public_html`, or any Apache/LAMP server.
