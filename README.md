# Karan Singh Thakur — Cyber Portfolio v2

This is the upgraded portfolio with the generated AI developer profile image, animated WebP profile loop, 3D tilt cards, matrix background, radar chart, terminal typewriter, glassmorphism UI, skill orbits, cursor glow, and project filtering.

## Run locally

Open `index.html` directly, or run:

```bash
python -m http.server 8000
```

Then open:

```text
http://localhost:8000
```

## Best hosting: GitHub Pages

1. Create a public GitHub repository:

```text
KaranSinghThakur700758703.github.io
```

2. Upload everything in this folder to the repository root.
3. Go to **Settings → Pages**.
4. Source: **Deploy from a branch**.
5. Branch: `main`, folder: `/root`.
6. Your portfolio will be live at:

```text
https://KaranSinghThakur700758703.github.io
```

## What to customize before publishing

- Replace the resume PDF in `assets/Karan-Singh-Thakur-Resume.pdf` with your latest Cogent/PHP resume.
- Add a real LinkedIn link in `index.html` if you want one in the header.
- Keep your GitHub repositories public and pin projects that match the portfolio case studies.
- Create GitHub repos for the displayed projects so the portfolio looks evidence-backed.
